# netrun

privacy-focused AI-native smart browser that i worked on all day for 38 straight days.

## Run

```bash
./setup.sh      # install dependencies (macOS, one time)
npm start       # launch the app
```

## Features

- Privacy hardening (HTTPS-only, tracker/cookie blocking, DoH, WebRTC leak guard)
- Ad & YouTube ad blocking
- AI chat, research, and browser agents (local Ollama or cloud OpenRouter)
- Nerd Mode — PDF / arXiv paper reader
- Personalized recommendation engine (arXiv, Hacker News, Polymarket)
- Encrypted password store
