Overlays

Modal surfaces that float above the main content — sheets, alerts, popovers, menus, and toast notifications.

Shared behavior All overlays mount to document.body, dismiss on backdrop click or Escape key, and integrate with the Motion animation system.

Sheet

Bottom drawer modal with backdrop. Animates up from the bottom with a spring curve.

const showSheet = State(false);

const sheet = Sheet(showSheet, () =>
  VStack(
    Text('Sheet Title').font('title2'),
    Text('Sheet content goes here.'),
    Button('Done').primary().onTap(() => sheet.dismiss())
  ).spacing(4).padding(6)
);

// Trigger
Button('Open Sheet').onTap(() => sheet.show())
MethodDescription
Sheet(isPresented, contentFn)Create a bottom sheet
.show()Present the sheet
.dismiss()Hide the sheet
.dispose()Remove from DOM and clean up

Alert

Centered modal dialog with title, message, and action buttons. Animates with a snappy scale spring.

const showAlert = State(false);

const alert = Alert(showAlert, {
  title: 'Delete Item?',
  message: 'This action cannot be undone.',
  actions: [
    { label: 'Cancel', style: 'cancel', handler: () => alert.dismiss() },
    { label: 'Delete', style: 'destructive', handler: () => deleteItem() }
  ]
});

Button('Delete').danger().onTap(() => alert.show())
MethodDescription
Alert(isPresented, opts)Create an alert dialog
.show()Present the alert
.dismiss()Hide the alert
.dispose()Clean up

Action Styles

StyleDescription
'primary'Primary action (filled button)
'destructive'Destructive action (red)
'cancel'Cancel action (ghost style)

Popover

Floating panel anchored to another view. Positions itself below the anchor with collision detection (flips above if needed).

const showPopover = State(false);
const anchor = Button('Info').ghost();

const popover = Popover(anchor, showPopover, () =>
  VStack(
    Text('Popover Content').font('subhead'),
    Text('Additional details here.').foreground('secondary')
  ).padding(4).frame({ width: 240 })
);

anchor.onTap(() => popover.show())
MethodDescription
Popover(anchor, isPresented, contentFn)Create a popover anchored to a view
.show()Present the popover
.dismiss()Hide the popover
.dispose()Clean up

Menu

Dropdown menu with support for anchor-toggle, context menu positioning, icons, dividers, custom views, and trailing content.

Anchor Menu

const btn = Button('Options').ghost();

const menu = Menu(btn, [
  { label: 'Edit', icon: '<svg>...</svg>', handler: () => edit() },
  { label: 'Duplicate', handler: () => duplicate() },
  { divider: true },
  { label: 'Delete', destructive: true, handler: () => remove() }
]);
// Menu toggles automatically on anchor click

Context Menu

const ctxMenu = Menu(null, [
  { label: 'Cut', handler: () => cut() },
  { label: 'Copy', handler: () => copy() },
  { label: 'Paste', handler: () => paste() }
]);

someView.on('contextmenu', e => {
  e.preventDefault();
  ctxMenu.showAt(e.clientX, e.clientY);
});

Menu Item Options

PropertyDescription
labelMenu item text

dividertrue for a separator line

MethodDescription
.show()Show menu below anchor
.showAt(x, y)Show at position (context menu)
.dismiss()Hide menu
.isOpenState signal for open/closed
.dispose()Clean up

Toast

Ephemeral notification at the bottom of the screen. Auto-dismisses after a duration.

// Direct creation
const t = Toast('File saved successfully');
t.dismiss(); // dismiss early

// With duration
Toast('Upload complete', { duration: 5000 });

// Global shortcut
showToast('Copied to clipboard');
SignatureDescription
Toast(message, opts?)Show a toast notification
showToast(message, opts?)Global alias (on window)

Options

OptionDefaultDescription
duration3000Auto-dismiss delay in milliseconds
