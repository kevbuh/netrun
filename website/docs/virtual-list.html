VirtualList

Virtualized scrolling for large datasets. Only renders the visible rows plus a configurable overscan buffer.

When to Use

  • Lists with hundreds or thousands of items
  • All rows have the same fixed height
  • You need smooth scrolling performance

For shorter lists (under ~100 items), use ForEach or List instead — the virtualization overhead isn't worth it for small datasets.

Basic Usage

const items = State(generateItems(10000));

VirtualList(
  items,
  item => item.id,       // key function (required)
  item => HStack(        // render function
    Text(item.name),
    Spacer(),
    Text(item.date).foreground('tertiary')
  ).padding(3),
  { rowHeight: 44, overscan: 5 }
).frame({ height: 400 })

How It Works

VirtualList maintains a pool of recycled DOM elements. On scroll, it calculates which rows are visible and repositions existing elements via transform: translateY().

  1. A sentinel element sets the total scrollable height based on items.length * rowHeight
  2. On scroll, the visible range is computed: start = max(0, floor(scrollTop / rowHeight) - overscan)
  3. Rows outside the range are returned to the pool
  4. New rows are taken from the pool or created fresh
  5. Each row is positioned absolutely via translateY

Performance Notes

  • Fixed row height required — all rows must be the same height for position calculations to work
  • Key function required — used to track which items map to which DOM elements
  • Overscan — higher values mean smoother fast scrolling but more DOM elements. 5 is a good default.
  • Reactive data — when the items signal changes, VirtualList resets and re-renders the visible range

Example with Reactive Data

const searchQuery = State('');
const allItems = State(loadItems());

const filtered = Computed(() => {
  const q = searchQuery.value.toLowerCase();
  if (!q) return allItems.value;
  return allItems.value.filter(i =>
    i.name.toLowerCase().includes(q)
  );
});

VStack(
  SearchField(searchQuery, 'Filter items...'),
  VirtualList(filtered, i => i.id, i =>
    Text(i.name).padding(3),
    { rowHeight: 40 }
  ).flex()
).spacing(3).frame({ height: '100%' })
