Primitives

Layout containers and basic views — the building blocks for composing interfaces.

VStack

Vertical flex column. Children stack top to bottom.

VStack(
  Text('First'),
  Text('Second'),
  Text('Third')
).spacing(3).alignment('center')
SignatureDescription
VStack(...children)Create a vertical stack
.spacing(s)Gap between children (space token)
.alignment(a)'center', 'leading', 'trailing', 'stretch'

HStack

Horizontal flex row. Children flow left to right. Vertically centered by default.

HStack(
  Icon('star', 14),
  Text('Starred'),
  Spacer(),
  Badge(5)
).spacing(2)
SignatureDescription
HStack(...children)Create a horizontal stack
.spacing(s)Gap between children (space token)
.alignment(a)'center', 'leading', 'trailing', 'stretch'

ZStack

Overlapping layer stack. First child defines size, others are positioned absolutely on top.

ZStack(
  Image('bg.jpg').fill(),
  VStack(
    Text('Overlay title').foreground('inverse')
  ).padding(4)
).alignment('bottomLeading')
ModifierDescription
.alignment(a)'center', 'top', 'bottom', 'leading', 'trailing', 'topLeading', 'topTrailing', 'bottomLeading', 'bottomTrailing'

Grid

CSS Grid container with convenience methods for columns, rows, and spacing.

// Fixed columns
Grid(card1, card2, card3, card4).columns(2).spacing(4)

// Auto-fill responsive columns
Grid(...items).columnWidth('200px').spacing(3)
ModifierDescription
.columns(n)repeat(n, 1fr)
.columnWidth(min)repeat(auto-fill, minmax(min, 1fr))
.rows(n)repeat(n, 1fr)
.spacing(s)Gap (both axes)
.rowSpacing(s)Row gap only
.columnSpacing(s)Column gap only
.alignment(a)Align items

ScrollView

Scrollable container. Vertical by default.

ScrollView(
  VStack(longListOfItems).spacing(2)
)

// Horizontal scrolling
ScrollView(
  HStack(thumbnails).spacing(3)
).horizontal()
ModifierDescription
.horizontal()Switch to horizontal scrolling

Spacer

Flexible whitespace that expands to fill available space. Use in HStack/VStack to push items apart.

HStack(
  Text('Left'),
  Spacer(),      // fills remaining space
  Text('Right')
)

Spacer(4) // minimum 16px (space token 4)

Divider

Horizontal separator line.

VStack(
  Text('Above'),
  Divider(),
  Text('Below')
).spacing(3)

Text

Renders text content. Accepts strings, numbers, signals, or functions for reactive text.

Text('Static text')
Text(count)                    // signal
Text(() => `Count: ${count.value}`)  // derived

Text('Styled')
  .font('headline')
  .bold()
  .italic()
  .mono()
  .selectable()
  .lineLimit(2)
  .align('center')
ModifierDescription
.bold()Font weight 600
.italic()Italic style
.mono()Monospace font
.code()Monospace + sunken background + padding
.align(a)Text alignment
.lineLimit(n)Clamp to n lines with ellipsis
.selectable()Allow text selection

Label

Icon + text combination in an HStack.

Label('Settings', 'gear')
Label('Downloads')  // text only, no icon

Link

Styled anchor element.

Link('Visit site', 'https://example.com')
Link(() => dynamicLabel.value)

Image

Image element with reactive src support.

Image('photo.jpg')
  .resizable()         // object-fit: contain
  .frame({ width: 200, height: 150 })

Image(urlSignal).fill()  // object-fit: cover
  .aspectRatio('16/9')
ModifierDescription
.resizable()object-fit: contain
.fill()object-fit: cover
.aspectRatio(r)CSS aspect-ratio value

Icon

Inline icon via the app's icon callback (window.icon()).

Icon('star', 16)
Icon('check').size(20).tint('accent')
ModifierDescription
.size(s)Icon size in pixels
.tint(color)Apply color token

Kbd

Keyboard shortcut display with automatic symbol mapping.

Kbd('Cmd+S')     // renders ⌘S
Kbd('Ctrl+Shift+P')  // renders ⌃⇧P

Symbol mapping: Cmd → ⌘, Ctrl → ⌃, Alt → ⌥, Shift → ⇧, Enter → ↵, Backspace → ⌫, Tab → ⇥, Esc → ⎋

RawHTML

Inject trusted HTML (e.g. SVG icons) as a View.

const icon = RawHTML('<svg viewBox="0 0 24 24">...</svg>');
HStack(icon, Text('Label'))
Security Only use RawHTML with trusted content. Never pass user input directly.
