Tokens & Theming

Design tokens are CSS custom properties with the --nr-* prefix. They provide a consistent design language across the entire application.

Spacing

Based on a 4px grid. Numeric values in modifiers like .padding() and .gap() resolve to these tokens.

Themes

Netrun includes four themes that override the --nr-* tokens:

  • Dark — default dark theme (defined in tokens.css)
  • Light — light background with dark text
  • Daylight — time-adaptive theme that shifts warmth throughout the day
  • Clear — glassmorphic theme with transparency and blur

Theme CSS files live in src/aether/css/themes/. Each file overrides the same --nr-* custom properties on :root, so the rest of the UI adapts automatically.

/* Example: light theme overrides */
:root {
  --nr-bg-body:    #ffffff;
  --nr-bg-surface: #f8f8f8;
  --nr-bg-raised:  #ffffff;
  --nr-text-primary: #111111;
  /* ... all other tokens ... */
}

Accent Colors

The accent color is set via --nr-accent and --nr-accent-hover. Users can choose from 8+ accent colors in settings.

// Using accent in code
Text('Important').foreground('accent')
Button('Action').primary() // uses accent as background
