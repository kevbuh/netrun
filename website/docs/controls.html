Controls

Interactive components for user input — buttons, text fields, toggles, sliders, and more. All controls support two-way binding via State or Binding.

Button

Button('Save').primary().onTap(() => save())
Button('Cancel').ghost()
Button('Delete').danger().small()
Button('').iconButton('plus')
Button('Export').icon('download', 'leading').large()
ModifierDescription
.primary()Primary variant
.secondary()Secondary variant
.ghost()Ghost/text variant
.danger()Destructive/red variant
.small()Small size
.large()Large size
.iconButton(name)Icon-only button
.icon(name, position?)Icon with label. Position: 'leading' or 'trailing'.

TextField

Text input with reactive two-way binding.

const name = State('');

TextField(name, 'Enter your name')
TextField('Just a placeholder')  // no binding
TextField(name).secure()         // password input
TextField(name).placeholder('Search...')
SignatureDescription
TextField(binding, placeholder?)Create text input. First arg: Signal/Binding or string placeholder.
.bind(b)Re-bind to a different signal
.secure()Password input type
.placeholder(p)Set placeholder text

Textarea

const bio = State('');

Textarea(bio, 'Tell us about yourself')
  .rows(5)
  .maxLength(500)
  .autoResize()
ModifierDescription
.rows(n)Set visible rows
.maxLength(n)Character limit
.autoResize()Auto-expand height as content grows

Toggle

Switch control with optional label.

const darkMode = State(false);

Toggle(darkMode, 'Dark mode')
Toggle(darkMode)  // no label

Checkbox

const agreed = State(false);

Checkbox(agreed, 'I agree to the terms')
Checkbox(agreed).indeterminate()
ModifierDescription
.indeterminate()Set indeterminate visual state

RadioGroup

const color = State('red');

RadioGroup(color, ['red', 'green', 'blue'])

// With value/label pairs
RadioGroup(color, [
  { value: 'red', label: 'Red' },
  { value: 'green', label: 'Green' },
  { value: 'blue', label: 'Blue' }
]).horizontal()
ModifierDescription
.horizontal()Horizontal layout
.options(opts)Replace options dynamically

Slider

const volume = State(50);

Slider(volume, { min: 0, max: 100, step: 1 })
Slider(volume).range(0, 100).step(5)
ModifierDescription
.range(min, max)Set min/max values
.step(s)Set step increment

Picker

Dropdown select control.

const lang = State('en');

Picker(lang, [
  { value: 'en', label: 'English' },
  { value: 'es', label: 'Spanish' },
  { value: 'fr', label: 'French' }
])

// Simple string options
Picker(lang, ['en', 'es', 'fr'])
ModifierDescription
.options(opts)Replace options dynamically

Stepper

Increment/decrement control with -/value/+ buttons.

const qty = State(1);

Stepper(qty, { min: 0, max: 99, step: 1 })

TabView

Tabbed content with cached tab panels. Tabs hide/show via display:none, not rebuild.

const activeTab = State('info');

TabView(activeTab, [
  { label: 'Info',     content: () => InfoPanel() },
  { label: 'Settings', content: () => SettingsPanel() },
  { label: 'About',    content: () => AboutPanel() }
])
ModifierDescription
.segmented()Segmented tab bar style
.underlined()Underlined tab bar style (default)
Caching TabView renders each tab once on first visit, then toggles visibility. Content functions are only called once per tab.

ProgressBar

const progress = State(0.6);

ProgressBar(progress).tint('accent')
ProgressBar(progress).indeterminate()
ModifierDescription
.tint(color)Fill bar color (token)
.indeterminate()Animated indeterminate state

Pill

Pill('New').accent()
Pill('Draft').outline()
Pill(count).small()
Pill('').dot()      // dot indicator
Pill('Tag').interactive().onTap(() => {})
ModifierDescription
.accent()Accent color style
.outline()Outline style
.small()Small size
.large()Large size
.dot()Dot indicator style
.interactive()Hoverable/clickable style

Badge

Notification badge. Auto-hides when value is 0, null, or undefined.

Badge(3)
Badge(unreadCount).tint('accent')
Badge('').dot()
ModifierDescription
.tint(color)Background color (token)
.dot()Dot style (no text)

SegmentedControl

const view = State('list');

SegmentedControl(view, ['list', 'grid', 'table'])
SegmentedControl(view, [
  { value: 'list', label: 'List' },
  { value: 'grid', label: 'Grid' }
])

FormField

Label + control + optional description/error wrapper.

const email = State('');

FormField('Email', TextField(email, 'you@example.com'), {
  description: 'We will never share your email.',
  error: 'Please enter a valid email.'
})

SearchField

Search input with icon, clear button, and built-in debounce.

const query = State('');

SearchField(query, 'Search documents...')
  .debounce(300)  // 300ms debounce
ModifierDescription
.debounce(ms)Debounce delay (default: 200ms)

Spinner

Spinner()     // 20px default
Spinner(32)   // 32px
Spinner().size(40)

Disclosure

Collapsible section with chevron toggle.

Disclosure('Advanced Settings', () =>
  VStack(
    Toggle(debug, 'Debug mode'),
    Toggle(verbose, 'Verbose logging')
  ).spacing(2)
).isExpanded(expanded).animates()
ModifierDescription
.isExpanded(signal)Control expanded state externally
.animates()Fade animation on expand

Skeleton

Loading placeholder.

Skeleton()                  // single line
Skeleton().lines(3)         // three lines
Skeleton().circle(48)       // circular avatar placeholder
ModifierDescription
.lines(n)Multiple line skeletons (default 3)
.circle(size?)Circle skeleton (default 40px)
