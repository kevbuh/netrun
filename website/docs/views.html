Views & Modifiers

Every UI element is a View. Modifiers configure appearance, layout, and behavior via fluent chaining. All modifiers return this.

View Base Class

View(tag) creates a DOM element. All primitives (VStack, Button, etc.) extend View. You rarely create a raw View directly.

// Access the live DOM element
const v = Text('Hello');
v.el; // HTMLSpanElement

// Append children after construction
const stack = VStack().add(
  Text('First'),
  Text('Second')
);

// Builder pattern
VStack().content(() => [
  Text('A'),
  Text('B')
]);

Core Methods

MethodDescription
.add(...children)Append child views/elements. Returns this.
.content(fn)Builder pattern — fn() returns array of children to append
.elThe live DOM element (already built at construction)
.dispose()Clean up effects, listeners, and children recursively
Convention Never call .build() or ._appendChildren() in consumer code. Use .el for the DOM element and .add() for children.

Mounting

// Mount — clears target, appends view, disposes previous
AetherUI.mount(view, document.getElementById('app'));

// Append — adds to target without clearing
AetherUI.append(view, document.getElementById('sidebar'));

// Serialize — walk view tree into semantic string (for AI context)
const desc = AetherUI.serialize(view);

Animation & Lifecycle

ModifierDescription
.animation(config)Deferred Motion animation (runs on appear)
.transition(props, duration?)CSS transition. props: array or string.
.onAppear(fn)Callback on mount. Stacks (multiple calls allowed).
.onDisappear(fn)Callback on unmount/dispose
Tip .onAppear() and .animation() stack — you can call them multiple times on the same view without overwriting previous callbacks.

Token Resolution

Numeric arguments to layout modifiers resolve to spacing tokens. String arguments pass through as-is:

// Token resolution
.padding(4)         // → var(--nr-space-4) = 16px
.padding('2rem')    // → 2rem (passthrough)
.cornerRadius('md') // → var(--nr-radius-md) = 8px
.background('surface') // → var(--nr-bg-surface)
.foreground('secondary') // → var(--nr-text-secondary)
.zIndex('overlay')  // → 1000
